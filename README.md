# Hangman Game 🎮

A simple Hangman game implemented in Python.  
Players try to guess the hidden word by suggesting letters within a limited number of attempts.

---

## 📂 Project Structure
- `hangman_01.py` → Main game logic  
- `hangman_stages.py` → ASCII art stages for the hangman  
- `.gitignore` → Keeps unnecessary files (like `__pycache__`) out of the repository
